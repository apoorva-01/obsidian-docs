# Marketing — obsidian-docs

This folder is the source-of-truth for marketing campaign work around `obsidian-docs`.
It is structured as an Obsidian-friendly mini-vault: open this repo in Obsidian and use this folder as your working space for campaigns, copy, and assets.

## Contents

- [[product-brief]] — what the product is, in one page
- [[features]] — shipped features + proposed roadmap (clearly separated)
- [[audience-and-positioning]] — who it's for, where they hang out, the core wedge
- [[messaging]] — taglines, value props, elevator pitches at 3 lengths
- [[campaign-ideas]] — concrete campaign concepts to design against
- [[assets-checklist]] — what we still need to produce (visuals, demos, etc.)

## How to use

1. Open the repo root in Obsidian (it is already an Obsidian-compatible folder structure once `obsidian-docs install` runs).
2. Edit notes here freely — nothing in this folder affects the CLI behavior.
3. Wikilinks (`[[note-name]]`) work between the notes in this folder.
